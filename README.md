# MyIO

m2yO (pronounced “my-eye-oh” or “my io”) is an experimental Rust web server project. This initial version implements a minimal blocking TCP listener that responds to basic HTTP requests. The project will evolve towards a lock-free, concurrent request processing architecture.

## Requirements

- [Rust](https://www.rust-lang.org/tools/install)
- [Cargo](https://doc.rust-lang.org/cargo/)
- [just](https://github.com/casey/just)
